# cpp-student-management
基于c++的简单学生成绩管理系统  

---
功能  
 * - [ ] 增删改  
 * - [ ] 按成绩排序  
 * - [ ] 按学号查找  
 * - [ ] 统计平均分、最高分  
 * - [ ] 从文件读写

---

算法  
 * - [ ] 排序算法
     - 按成绩排序
     - 按多关键字排序
 * - [ ] 二分查找
     - 学号已排序->快速查找学生
 * - [ ] 前缀和/统计类算法
     - 场景区间统计
     - 分数段人数统计
 * - [ ] 离线查询
     - 查询多个成绩区间的人数
     - 查询某段学号区间的平均分
 * - [ ] 哈希/映射
     - 学号->学生映射
     - 姓名->学生列表
 * - [ ] Top-K/选择算法
     - 成绩最高的前k名
 * - [ ] 动态维护排名
 * - [ ] 区间问题
